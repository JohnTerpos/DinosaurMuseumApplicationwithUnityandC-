# DinosaurMuseumApplicationwithUnityandC#
Στην εφαρμογή, παρουσιάζεται ένα μουσείο δεινοσαύρων. Αυτό το μουσείο περιέχει μεγάλη ποικιλία δεινοσαύρων όπου περικλείονται βασικά χαρακτηριστικά τους καθώς και διάφορα animations που τους προστέθηκαν. Επιπλέον, συμπεριλαμβάνεται και ένα μαγαζί souvenirs όπου ο χρήστης μπορεί να αγοράσει χρησιμοποιώντας πιστωτική κάρτα.
